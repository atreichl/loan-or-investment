Since money and management amd investing are important for long term stability. I had the question of if I should put any extra income I have towards paying off my student loans first
or should I start investest that money right away.

So I created this web application that allows a user to enter investment and loan information. 
It then calculates the information for using any flexable money you have for paying off your loan first then investing it or investing right away.
I also add an option to move a percentage of you loan payments to investments once you have finished pay off your loan.

It is not hosted on a server so to use this tool you need to download it and run it with py manage.py runserver and then connect to the local host.
